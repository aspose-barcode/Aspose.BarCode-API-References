## BarCodeGeneratorControl.ThrowExceptionWhenCodeTextIncorrect property

Μόνο για γραμμωτούς κώδικες 1D. Εάν το κωδικοποιημένο κείμενο είναι λανθασμένο και η τιμή οριστεί σε true - θα γίνει εξαίρεση. Διαφορετικά, το κωδικοποιημένο κείμενο θα διορθωθεί ώστε να ταιριάζει με την προδιαγραφή του γραμμικού κώδικα. Θα γίνεται πάντα εξαίρεση για: Συμβολολογία γραμμής δεδομένων εάν το κωδικοποιημένο κείμενο είναι λανθασμένο. Η εξαίρεση δεν θα γίνεται πάντα για: AustraliaPost, SingapurePost, Code39Extended, Code9381Extended Coderectext, .

public bool ThrowExceptionWhenCodeTextIncorrect { get; set; }
