## IBarCodeGeneratorControl.RotationAngle property

Γωνία περιστροφής εικόνας BarCode, μετρημένη σε μοίρες, π.χ. Γωνία Περιστροφής = 0 ή Γωνία Περιστροφής = 360 σημαίνει ότι δεν υπάρχει περιστροφή. Εάν η γωνία περιστροφής ΔΕΝ ισούται με 90, 180, 270 ή 0, μπορεί να αυξήσει τη δυσκολία του σαρωτή να διαβάσει__x00 την εικόνα.
