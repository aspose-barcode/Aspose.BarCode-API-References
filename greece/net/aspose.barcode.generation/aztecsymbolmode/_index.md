## AztecSymbolMode enumeration

Καθορίζει τη λειτουργία συμβόλων των Αζτέκων.

```csharp
public enum AztecSymbolMode
```

### Αξίες

| Ονομα | αξία | Περιγραφή |
| --- | --- | --- |
| Auto | `0` | Καθορίζει την αυτόματη λήψη του καλύτερου συμβόλου (Συμπαγές ή Πλήρους εύρους) για Αζτέκους. Αυτή είναι η προεπιλεγμένη τιμή. |
| Compact | `1` | Καθορίζει το σύμβολο Compact για τους Αζτέκους. Το σύμβολο Aztec Compact επιτρέπει μόνο 1, 2, 3 ή 4 επίπεδα. |
| FullRange | `2` | Καθορίζει το σύμβολο πλήρους εύρους για τους Αζτέκους. Το σύμβολο πλήρους εύρους των Αζτέκων επιτρέπει από 1 έως 32 επίπεδα. |
| Rune | `3` | Καθορίζει το σύμβολο Rune για τους Αζτέκους. Οι Ρούνοι των Αζτέκων είναι μια σειρά από μικρά αλλά διακριτά σήματα αναγνώσιμα από μηχανή. Επιτρέπει μόνο αριθμητική τιμή από 0 έως 255. |

### Παραδείγματα

Αυτό το δείγμα δείχνει πώς να αλλάξετε τη λειτουργία συμβόλων των Αζτέκων και να αποθηκεύσετε μια εικόνα BarCode.

```csharp
[C#]
using (Aspose.BarCode.Generation.BarcodeGenerator generator = new Aspose.BarCode.Generation.BarcodeGenerator(EncodeTypes.Aztec))
```
