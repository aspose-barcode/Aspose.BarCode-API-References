## BarCodeConfidence enumeration

Περιέχει επίπεδο εμπιστοσύνης αναγνώρισης

```csharp
public enum BarCodeConfidence
```

### Αξίες

| Ονομα | αξία | Περιγραφή |
| --- | --- | --- |
| None | `0` | Εμπιστοσύνη αναγνώρισης γραμμικού κώδικα όπου το κωδικοποιημένο κείμενο δεν αναγνωρίστηκε σωστά ή ο γραμμωτός κώδικας εντοπίστηκε ως πιθανός fake |
| Moderate | `80` | Εμπιστοσύνη αναγνώρισης γραμμωτού κώδικα (κυρίως γραμμωτοί κώδικες 1D) με αδύναμο άθροισμα ελέγχου ή ακόμα και χωρίς αυτό. Θα μπορούσε να περιέχει κάποιες εσφαλμένες αναγνωρίσεις στο codetext ή ακόμα και ψευδείς αναγνωρίσεις εάν είναι low |
| Strong | `100` | Εμπιστοσύνη αναγνώρισης που επιβεβαιώθηκε με κωδικούς BCH όπως Reed–Solomon. Δεν πρέπει να υπάρχουν σφάλματα στην ανάγνωση κωδικοποιημένου κειμένου ή ψευδείς αναγνωρίσεις |

[C#]
//Μέτρια εμπιστοσύνη
using (BarcodeGenerator generator = new BarcodeGenerator(EncodeTypes.Code128, "12345"))
{
    generator.Save(@"c:\test.png");
}
using (BarCodeReader reader = new BarCodeReader(@"c:\test.png", DecodeType.Code39Standard, DecodeType.Code128))
{
    foreach (BarCodeResult result in reader.ReadBarCodes())
    {
        Console.WriteLine("BarCode Type: " + result.CodeTypeName);
        Console.WriteLine("BarCode CodeText: " + result.CodeText);
        Console.WriteLine("BarCode Confidence: " + result.Confidence);
        Console.WriteLine("BarCode ReadingQuality: " + result.ReadingQuality);
    }
}
