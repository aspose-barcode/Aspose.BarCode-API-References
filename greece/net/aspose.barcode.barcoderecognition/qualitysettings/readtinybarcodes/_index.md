## QualitySettings.ReadTinyBarcodes property

Επιτρέπει στον κινητήρα να αναγνωρίζει μικροσκοπικούς γραμμωτούς κώδικες σε μεγάλες εικόνες. Αγνοήθηκε αν[`AllowIncorrectBarcodes`](../allowincorrectbarcodes/) έχει οριστεί σε True. Προεπιλεγμένη τιμή: False.

```csharp
public bool ReadTinyBarcodes { get; set; }
```

### Αξία περιουσίας

Εάν είναι True, επιτρέπει στον κινητήρα να αναγνωρίζει μικροσκοπικούς γραμμωτούς κώδικες σε μεγάλες εικόνες.
