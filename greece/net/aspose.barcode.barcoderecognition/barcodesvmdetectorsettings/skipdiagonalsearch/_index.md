## BarcodeSvmDetectorSettings.SkipDiagonalSearch property

Επιτρέπει στον ανιχνευτή να παραλείψει την αναζήτηση για διαγώνιους γραμμικούς κώδικες.

Η ρύθμιση του σε false θα αυξήσει τον χρόνο ανίχνευσης, αλλά θα επιτρέψει την εύρεση διαγώνιων γραμμωτών κωδίκων που μπορεί να χαθούν διαφορετικά. Η ενεργοποίηση της διαγώνιας αναζήτησης οδηγεί σε μεγαλύτερο χρόνο ανίχνευσης.

```csharp
public bool SkipDiagonalSearch { get; set; }
```
