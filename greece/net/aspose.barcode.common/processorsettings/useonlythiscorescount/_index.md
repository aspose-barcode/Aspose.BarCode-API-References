## ProcessorSettings.UseOnlyThisCoresCount property

Καθορίστε τον αριθμό των πυρήνων που θα χρησιμοποιηθούν. Πρέπει να αλλάξετε την ιδιότητα "UseAllCores" σε "false".

```csharp
public int UseOnlyThisCoresCount { get; set; }
```
