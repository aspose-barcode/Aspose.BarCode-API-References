## SwissQRBill.Reference property

Λαμβάνει ή ορίζει την αναφορά πληρωμής πιστωτή.

Η αναφορά είναι υποχρεωτική για SwissQR IBAN, δηλ. IBAN στην περιοχή CHxx30000xxxxxx έως CHxx31999xxxxx.

Εάν ορίζεται, η αναφορά πρέπει να είναι είτε έγκυρη SwissQR reference (που αντιστοιχεί στο έντυπο αναφοράς ISR) είτε έγκυρη αναφορά πιστωτή σύμφωνα με το ISO 11649 ("RFxxxx"). Και τα δύο μπορεί να περιέχουν κενά για μορφοποίηση.

public string Reference { get; set; }
