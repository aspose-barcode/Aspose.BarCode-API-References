BarcodeScanner-inställningarna. Innehåller parametrar som påverkar igenkänningsprocessen och de igenkända data. BarcodeScannerPreferences ärver BarcodeRecognitionSettings, BarcodeScannerPreferences-klassen innehåller specifika funktioner för BarcodeScanner API och implementerar Parcelable-gränssnittet för att utbyta inställningar mellan klientaktivitet och BarcodeScannerActivity. BarcodeScannerPreferences fungerar endast inom klientapplikationen via BarcodeScannerActivity
