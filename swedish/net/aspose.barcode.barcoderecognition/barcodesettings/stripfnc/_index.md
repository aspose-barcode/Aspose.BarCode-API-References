## BarcodeSettings.StripFNC property

Ta bort FNC1, FNC2, FNC3 tecken från kodtext. Standardvärdet är false.

```csharp
public bool StripFNC { get; set; }
```

### Fastighetsvärde

Ta bort FNC1, FNC2, FNC3 tecken från kodtext. Standardvärdet är falskt.

### Exempel

Detta exempel visar hur man tar bort FNC-tecken

```csharp
[C#]
using (BarcodeGenerator generator = new BarcodeGenerator(EncodeTypes.GS1Code128, "(02)04006664241007(37)1(400)7019590754"))
{
    generator.Save(@"c:\test.png");
}
using (BarCodeReader reader = new BarCodeReader(@"c:\test.png", DecodeType.Code128))
{
    //StripFNC inaktiverad
    reader.BarcodeSettings.StripFNC = false;
    foreach (BarCodeResult result in reader.ReadBarCodes())
    {
        Console.WriteLine("BarCode CodeText: " + result.CodeText);
    }
}
using (BarCodeReader reader = new BarCodeReader(@"c:\test.png", DecodeType.Code128))
{
    //StripFNC aktiverad
    reader.BarcodeSettings.StripFNC = true;
    foreach (BarCodeResult result in reader.ReadBarCodes())
    {
        Console.WriteLine("BarCode CodeText: " + result.CodeText);
    }
}
[VB.NET]
```
