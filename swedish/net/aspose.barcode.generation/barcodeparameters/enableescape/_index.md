## BarcodeParameters.EnableEscape property

Anger om tecknet "\" förklaras som ett escape-tecken i CodeText-egenskapen. Används för Pdf417, DataMatrix, Code128 only Om EnableEscape är sant, kommer "\" att förklaras som ett speciellt escape-tecken. Annars fungerar "\" som vanliga tecken. Aspose.BarCode stöder inmatning av decimal ascii-kod och mnemonic för ASCII-kontrollkodstecken. Till exempel, \013 och \\CR står för CR.
