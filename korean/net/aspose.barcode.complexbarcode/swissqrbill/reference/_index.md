## SwissQRBill.Reference property

채권자 지불 참조를 가져오거나 설정합니다.

SwissQR IBAN, 즉 CHxx30000xxxxxx ~ CHxx31999xxxxx. 범위의 IBAN에 대한 참조는 필수입니다.

지정된 경우 참조는 ISO 11649("RFxxxx")에 따라 유효한 SwissQR 참조 (ISR 참조 양식에 해당) 또는 유효한 채권자 참조 여야 합니다. 둘 다 서식 지정을 위한 공백을 포함할 수 있습니다.

```csharp
public string Reference { get; set; }
```
