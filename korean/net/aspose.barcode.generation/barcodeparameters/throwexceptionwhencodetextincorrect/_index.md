## BarcodeParameters.ThrowExceptionWhenCodeTextIncorrect property

1D 바코드에만 해당. 코드 텍스트가 올바르지 않고 값이 true로 설정된 경우 예외가 발생합니다. 그렇지 않으면 코드 텍스트가 바코드 사양과 일치하도록 수정됩니다. 코드 텍스트가 잘못된 경우 데이터바 기호에 대해 항상 예외가 발생합니다. 코드 텍스트가 잘못된 경우 AustraliaPost, SingapurePost, Code39Extended, Code93Extended, Code16K, Code128 기호에 대해 예외가 항상 발생하지 않습니다. .
