## PatchFormat enumeration

PatchCode 형식. 단일 PatchCode를 생성하려면 PatchOnly를 선택합니다. 페이지 형식을 사용하여 PatchCodes가 borders 인 패치 페이지를 생성합니다.

```csharp
public enum PatchFormat
```

### 가치

| 이름 | 값 | 설명 |
| --- | --- | --- |
| PatchOnly | `0` | 패치 코드만 생성 |
| A4 | `1` | 테두리로 PatchCodes가 있고 center 에 선택적 QR이 있는 A4 형식 페이지를 생성합니다. |
| A4_LANDSCAPE | `2` | 테두리로 PatchCodes가 있고 center 에 선택적 QR이 있는 A4 가로 형식 페이지를 생성합니다. |
| US_Letter | `3` | 테두리로 PatchCode가 있고 center 에 선택적 QR이 있는 US 문자 형식 페이지를 생성합니다. |
| US_Letter_LANDSCAPE | `4` | 테두리로 PatchCode를 사용하고 center 에 선택적 QR을 사용하여 미국 문자 가로 형식 페이지를 생성합니다. |
