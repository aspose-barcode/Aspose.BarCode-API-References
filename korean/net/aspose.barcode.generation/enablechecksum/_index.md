## EnableChecksum enumeration

1D 바코드 생성 중 체크섬을 활성화합니다.

기본값은 체크섬을 포함해야 하는 심볼로지의 경우 Yes로 처리되며 체크섬만 가능한 경우 No로 처리됩니다.

사용되지 않은 체크섬: Codabar

체크섬 가능: Code39 Standard/Extended, Standard2of5, Interleaved2of5, Matrix2of5, ItalianPost25, DeutschePostIdentcode, DeutschePostLeitcode, VIN

항상 사용되는 체크섬: 나머지 기호

```csharp
public enum EnableChecksum
```

### 가치

| 이름 | 값 | 설명 |
| --- | --- | --- |
| Default | `0` | 사양에 체크섬이 필요한 경우 첨부됩니다. |
| Yes | `1` | 가능하면 항상 체크섬을 사용하십시오. |
| No | `2` | 체크섬을 사용하지 마십시오. |
