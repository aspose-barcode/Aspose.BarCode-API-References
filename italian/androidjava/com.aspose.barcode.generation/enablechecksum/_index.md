```
public enum EnableChecksum extends Enum<EnableChecksum>
```

Abilita il checksum durante la generazione per i codici a barre 1D.

Il valore predefinito è considerato Sì per le simbologie che devono contenere il checksum, e No dove il checksum è solo opzionale.

Checksum mai usato: Codabar

Checksum è possibile: Code39 Standard/Extended, Standard2of5, Interleaved2of5, Matrix2of5, ItalianPost25, DeutschePostIdentcode, DeutschePostLeitcode, VIN

Checksum sempre usato: Resto delle simbologie
## Campi

| Campo | Descrizione |
| --- | --- |
| [DEFAULT](#DEFAULT) | Se il checksum è richiesto dalla specifica - verrà allegato. |
| [NO](#NO) | Non utilizzare il checksum. |
| [YES](#YES) | Usa sempre il checksum se possibile. |
