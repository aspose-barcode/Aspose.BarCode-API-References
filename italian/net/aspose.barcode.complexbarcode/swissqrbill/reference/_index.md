## SwissQRBill.Reference property

Ottiene o imposta il riferimento di pagamento del creditore.

Il riferimento è obbligatorio per gli IBAN SwissQR, ovvero gli IBAN nell'intervallo da CHxx30000xxxxxx a CHxx31999xxxxx.

Se specificato, il riferimento deve essere un riferimento SwissQR valido (corrispondente al modulo di riferimento PVR) o un riferimento creditore valido secondo ISO 11649 ("RFxxxx"). Entrambi possono contenere spazi per la formattazione.

```csharp
public string Reference { get; set; }
```

### Valore della proprietà

Il riferimento di pagamento del creditore.
