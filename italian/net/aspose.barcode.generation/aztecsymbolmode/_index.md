## AztecSymbolMode enumeration

Specifica la modalità del simbolo azteco.

```csharp
public enum AztecSymbolMode
```

### I valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| Auto | `0` | Specifica di raccogliere automaticamente il miglior simbolo (Compatto o Intero) per Aztec. Questo è il valore predefinito. |
| Compact | `1` | Specifica il simbolo Compact per Aztec. Il simbolo Aztec Compact consente solo 1, 2, 3 o 4 livelli. |
| FullRange | `2` | Specifica il simbolo di intervallo completo per Aztec. Il simbolo di intervallo completo azteco consente da 1 a 32 livelli. |
| Rune | `3` | Specifica il simbolo della runa per Aztec. Le rune azteche sono una serie di segni piccoli ma distinti leggibili dalla macchina. Consente solo valori numerici da 0 a 255. |

### Esempi

Questo esempio mostra come modificare la modalità Simbolo azteco e salvare un'immagine di codice a barre.

```csharp
[C#]
using (Aspose.BarCode.Generation.BarcodeGenerator generator = new Aspose.BarCode.Generation.BarcodeGenerator(EncodeTypes.Aztec))
{
    generator.CodeText = "125";
    generator.Parameters.Barcode.Aztec.AztecSymbolMode = AztecSymbolMode.Rune;
    generator.Save("test.png");
}
[VB.NET]
Using generator As New Aspose.BarCode.Generation.BarcodeGenerator(EncodeTypes.Aztec)
    generator.CodeText = "125"
    generator.Parameters.Barcode.Aztec.AztecSymbolMode = AztecSymbolMode.Rune
    generator.Save("test.png")
End Using
```
