## BaseGenerationParameters.RotationAngle property

Angolo di rotazione dell'immagine BarCode, misurato in gradi, es. RotationAngle = 0 o RotationAngle = 360 significa nessuna rotazione. Se RotationAngle NON è uguale a 90, 180, 270 o 0, può aumentare la difficoltà per lo scanner di leggere l'immagine. Valore predefinito: 0.

```csharp
public float RotationAngle { get; set; }
```

[C#]
