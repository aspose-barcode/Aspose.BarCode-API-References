## BarCodeConfidence enumeration

Contiene il livello di affidabilità del riconoscimento

```csharp
public enum BarCodeConfidence
```

### I valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| None | `0` | Fiducia di riconoscimento del codice a barre in cui il testo del codice non è stato riconosciuto correttamente o il codice a barre è stato rilevato come possibile falso |
| Moderate | `80` | Fiducia di riconoscimento del codice a barre (principalmente codici a barre 1D) con checksum debole o anche senza di esso. Potrebbe contenere alcuni riconoscimenti errati nel codetext o anche falsi riconoscimenti se è basso |
| Strong | `100` | Fiducia del riconoscimento che è stata confermata con codici BCH come Reed–Solomon. Non devono esserci errori nella lettura del testo del codice o falsi riconoscimenti |

Questo esempio mostra come è cambiato BarCodeConfidence, a seconda del tipo di codice a barre

    generator.Save(@"c:\test.png");

{
    foreach (BarCodeResult result in reader.ReadBarCodes())
    {
        Console.WriteLine("BarCode Type: " + result.CodeTypeName);
        Console.WriteLine("BarCode CodeText: " + result.CodeText);
        Console.WriteLine("BarCode Confidence: " + result.Confidence);
        Console.WriteLine("BarCode ReadingQuality: " + result.ReadingQuality);
    }
}
