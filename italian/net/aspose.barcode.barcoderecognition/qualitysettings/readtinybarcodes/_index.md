## QualitySettings.ReadTinyBarcodes property

Consente al motore di riconoscere codici a barre minuscoli su immagini di grandi dimensioni. Ignorato se[`AllowIncorrectBarcodes`](../allowincorrectbarcodes) è impostato su Vero. Valore predefinito: False.

```csharp
public bool ReadTinyBarcodes { get; set; }
```

### Valore della proprietà

Se True, consente al motore di riconoscere codici a barre minuscoli su immagini di grandi dimensioni.
