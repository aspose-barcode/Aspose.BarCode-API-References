## QualitySettings.ReadTinyBarcodes property

Permet au moteur de reconnaître les minuscules codes-barres sur les grandes images. Ignoré si[`AllowIncorrectBarcodes`](../allowincorrectbarcodes) est défini sur Vrai. Valeur par défaut : Faux.

```csharp
public bool ReadTinyBarcodes { get; set; }
```

### Valeur de la propriété

Si True, permet au moteur de reconnaître les minuscules codes-barres sur les grandes images.
