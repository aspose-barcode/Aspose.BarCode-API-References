## QualitySettings.AllowDecreasedImage property

Permet au moteur de reconnaître une image diminuée en tant que numérisation supplémentaire. La taille pour la diminution est sélectionnée par les algorithmes du moteur interne. Le mode aide à reconnaître les codes-barres qui sont bruyants et flous mais capturés avec une haute résolution.

```csharp
public bool AllowDecreasedImage { get; set; }
```

### Valeur de la propriété

Permet au moteur de reconnaître une image réduite
