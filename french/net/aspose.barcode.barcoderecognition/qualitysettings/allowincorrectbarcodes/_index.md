## QualitySettings.AllowIncorrectBarcodes property

Permet au moteur de reconnaître les codes à barres qui ont une somme de contrôle incorrecte ou des valeurs incorrectes. Le mode peut être utilisé pour reconnaître les codes-barres endommagés avec un texte incorrect.

```csharp
public bool AllowIncorrectBarcodes { get; set; }
```

### Valeur de la propriété
