## BarCodeConfidence enumeration

Contient le niveau de confiance de la reconnaissance

```csharp
public enum BarCodeConfidence
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| None | `0` | Fiabilité de la reconnaissance du code-barres lorsque le texte du code n'a pas été reconnu correctement ou que le code-barres a été détecté comme un faux possible |
| Moderate | `80` | Confiance de la reconnaissance des codes-barres (principalement des codes-barres 1D) avec une somme de contrôle faible ou même sans. Peut contenir des erreurs de reconnaissance dans codetext ou même de fausses reconnaissances si est faible |
| Strong | `100` | Confiance de reconnaissance qui a été confirmée avec des codes BCH comme Reed–Solomon. Il ne doit pas y avoir d'erreurs dans le texte de code lu ou de fausses reconnaissances |

### Exemples

Cet exemple montre comment BarCodeConfidence a changé, en fonction du type de code-barres

```csharp
[C#]
//Confiance modérée
using (BarcodeGenerator generator = new BarcodeGenerator(EncodeTypes.Code128, "12345"))
{
    generator.Save(@"c:\test.png");
}
using (BarCodeReader reader = new BarCodeReader(@"c:\test.png", DecodeType.Code39Standard, DecodeType.Code128))
{
    foreach (BarCodeResult result in reader.ReadBarCodes())
    {
        Console.WriteLine("BarCode Type: " + result.CodeTypeName);
        Console.WriteLine("BarCode CodeText: " + result.CodeText);
        Console.WriteLine("BarCode Confidence: " + result.Confidence);
        Console.WriteLine("BarCode ReadingQuality: " + result.ReadingQuality);
    }
}

// Forte confiance
using (BarcodeGenerator generator = new BarcodeGenerator(EncodeTypes.QR, "12345"))
{
    generator.Save(@"c:\test.png");

        Console.WriteLine("BarCode CodeText: " + result.CodeText);
        Console.WriteLine("BarCode Confidence: " + result.Confidence);

    }
}
```
