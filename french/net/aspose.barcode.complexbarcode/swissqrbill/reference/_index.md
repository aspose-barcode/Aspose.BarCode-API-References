## SwissQRBill.Reference property

Obtient ou définit la référence de paiement du créancier.

La référence est obligatoire pour les IBAN SwissQR, c'est-à-dire les IBAN dans la plage CHxx30000xxxxxx à CHxx31999xxxxx.

Si spécifié, la référence doit être soit une référence SwissQR valide (correspondant au formulaire de référence BVR) soit une référence créancier valide selon ISO 11649 ("RFxxxx"). Les deux peuvent contenir des espaces pour le formatage.

```csharp
public string Reference { get; set; }
```

### Valeur de la propriété

La référence de paiement du créancier.
