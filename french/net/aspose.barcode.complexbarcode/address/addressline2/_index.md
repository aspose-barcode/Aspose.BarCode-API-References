## Address.AddressLine2 property

Obtient ou définit la ligne d'adresse 2.

La ligne d'adresse 2 contient le code postal et la ville.

La définition de ce champ définit le type d'adresse surCombinedElements sauf si c'est déjà Structured , auquel cas il devientConflicting .

Ce champ n'est utilisé que pour les adresses d'éléments combinés. Pour ce type, c'est obligatoire.

```csharp
public string AddressLine2 { get; set; }
```

### Valeur de la propriété

La ligne d'adresse 2.
