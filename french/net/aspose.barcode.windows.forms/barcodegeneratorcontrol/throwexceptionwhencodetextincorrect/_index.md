## BarCodeGeneratorControl.ThrowExceptionWhenCodeTextIncorrect property

Uniquement pour les codes-barres 1D. Si le texte du code est incorrect et que la valeur est définie sur true - une exception sera levée. Sinon, le texte de code sera corrigé pour correspondre à la spécification du code-barres. L'exception sera toujours levée pour : la symbologie de la barre de données si le texte de code est incorrect. L'exception ne sera toujours pas levée pour : AustraliaPost, SingapurePost, Code39Extended, Code93Extended, Code16K, la symbologie Code128 si le texte de code est incorrect .
