## EncodeTypes.UpcaGs1Code128Coupon field

Spécifie que les données doivent être encodées avec **Coupon UPC avec code étendu GS1-128** spécification de code à barres. Exemple de chaîne d'entrée : BarcodeGenerator.Codetext = "514141100906(8102)03", où la partie UPCA est "514141100906", la partie GS1Code128 est (8102)03.
