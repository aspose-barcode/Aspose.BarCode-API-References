## EncodeTypes.UpcaGs1DatabarCoupon field

Spécifie que les données doivent être encodées avec **Coupon UPC avec ajout de GS1 DataBar**barcode specification. An example of the input string: BarcodeGenerator.Codetext = "514141100906(8110)106141416543213500110000310123196000", where UPCA part is "514141100906", Databar part is "(8110)106141416543213500110000310123196000".
