## QREncodeType enumeration

Mode sélecteur QR / MicroQR. Sélectionnez ForceQR pour les symboles QR standard, Auto pour MicroQR. ForceMicroQR est utilisé pour la génération de symboles fortement MicroQR si cela est possible.

```csharp
public enum QREncodeType
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| Auto | `0` | Le mode démarre la négociation de la version du code-barres à partir de MicroQR V1 |
| ForceQR | `1` | Le mode démarre la négociation de la version du code-barres à partir de QR V1 |
| ForceMicroQR | `2` | Le mode démarre la négociation de la version du code-barres de MicroQR V1 à V4. Si les données ne peuvent pas être encodées dans MicroQR, une exception est levée. |
