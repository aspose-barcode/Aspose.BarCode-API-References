## QREncodeMode enumeration

Mode d'encodage pour les codes-barres QR. Il est recommandé d'utiliser Auto avec CodeTextEncoding = Encoding.UTF8 pour les symboles ou chiffres latins et Utf8BOM pour les symboles Unicode.

```csharp
public enum QREncodeMode
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| Auto | `0` | Encoder le texte de code comme un jeu de caractères non Unicode. S'il y a un caractère Unicode, le texte de code sera encodé avec la valeur définie dans CodeTextEncoding. |
| Bytes | `1` | Encoder le texte de code sous forme d'octets simples. S'il détecte un caractère Unicode, le caractère sera codé sur deux octets, l'octet inférieur en premier. |
| Utf8BOM | `2` | Encoder le codetext avec l'encodage UTF8 avec le premier caractère ByteOfMark. |
| Utf16BEBOM | `3` | Encoder le codetext avec l'encodage UTF8 avec le premier caractère ByteOfMark. Il peut y avoir des problèmes avec certains lecteurs de codes-barres. |
| ECIEncoding | `4` | Codetext avec la valeur définie dans la propriété ECIEncoding. Il peut y avoir des problèmes avec certains anciens lecteurs de codes-barres (avant 2006). |
| ExtendedCodetext | `5` | Mode canal étendu qui prend en charge la première position FNC1, la deuxième position FNC1 et les modes multi-ECI. |
