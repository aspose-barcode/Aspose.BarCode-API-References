## MacroCharacter enumeration

Les valeurs des caractères macro 05 et 06 sont utilisées pour obtenir un codage plus compact dans des modes spéciaux. 05 Le caractère macro est traduit par "[)&gt;\u001E05\u001D" comme en-tête de données décodées et "\u001E\u0004" comme fin de données décodées. 06 Le caractère de macro est traduit en "[)&gt;\u001E06\u001D" comme en-tête de données décodées et "\u001E\u0004" comme fin de données décodée.

| Nom | Évaluer | La description |
| --- | --- | --- |
| None | `0` | Aucun des caractères de macro n'est ajouté aux données de code-barres |
| Macro05 | `5` | 05 Macro craracter est ajouté aux données du code-barres en première position. GS1 Data Identifier ISO 15434 Le caractère est traduit par "[)&gt;\u001E05\u001D" comme en-tête de données décodées et "\u001E\u0004" comme fin de données décodées. |
| Macro06 | `6` | 06 Macro craracter est ajouté aux données du code-barres en première position. ASC MH10 Data Identifier ISO 15434 Le caractère est traduit en "[)&gt;\u001E06\u001D" comme en-tête de données décodées et "\u001E\u0004" comme fin de données décodées. |
