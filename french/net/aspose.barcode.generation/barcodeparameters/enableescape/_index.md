## BarcodeParameters.EnableEscape property

Indique si explique le caractère "\" comme un caractère d'échappement dans la propriété CodeText. Utilisé pour Pdf417, DataMatrix, Code128 uniquement Si EnableEscape est vrai, "\" sera expliqué comme un caractère d'échappement spécial. Sinon, "\" agit comme des caractères normaux. Aspose.BarCode prend en charge la saisie de code ascii décimal et de mnémonique pour les caractères de code de contrôle ASCII. Par exemple, \013 et \\CR signifie CR.
