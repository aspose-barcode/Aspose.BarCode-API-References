## AutoSizeMode enumeration

Spécifie les différents types de modes de dimensionnement automatique.

```csharp
public enum AutoSizeMode
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| None | `0` | Le redimensionnement automatique est désactivé. |
| Nearest | `1` | Redimensionne le code-barres à la taille la plus basse possible spécifiée par les propriétés BarCodeWidth et BarCodeHeight. Préserve le format d'image par défaut. |
