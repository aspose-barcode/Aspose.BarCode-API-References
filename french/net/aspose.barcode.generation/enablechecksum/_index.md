## EnableChecksum enumeration

Activer la somme de contrôle lors de la génération pour les codes-barres 1D.

La valeur par défaut est traitée comme Oui pour les symbologies qui doivent contenir une somme de contrôle, comme Non lorsque seule la somme de contrôle est possible.

Somme de contrôle jamais utilisée : Codabar

La somme de contrôle est possible : Code39 Standard/Extended, Standard2of5, Interleaved2of5, Matrix2of5, ItalianPost25, DeutschePostIdentcode, DeutschePostLeitcode, VIN

Somme de contrôle toujours utilisée : symbologies de repos


### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| Default | `0` | Si la somme de contrôle est requise par la spécification - elle sera jointe. |
| Yes | `1` | Toujours utiliser la somme de contrôle si possible. |
| No | `2` | Ne pas utiliser la somme de contrôle. |
