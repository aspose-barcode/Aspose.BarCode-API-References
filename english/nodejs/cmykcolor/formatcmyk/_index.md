## formatCMYK() {#formatcmyk}

Format this CMYKColor into a string "C_M_Y_K", multiplying each internal component (0–1) by 100 and rounding.

```javascript
formatCMYK()
```

### Return Value

string

e.g. "30_100_0_30"
