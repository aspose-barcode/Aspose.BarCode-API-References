## Code128EncodeMode enumeration

Encoding mode for Code128 barcodes. `Code 128` specification.

```javascript
public enum Code128EncodeMode
```

## Values

| Name | Value | Description |
| --- | :---: | --- |
| AUTO | `0` | Encode codetext in classic ISO 15417 mode. The mode should be used in all ordinary cases. |
| CODE_A | `1` | Encode codetext only in 128A codeset. |
| CODE_B | `2` | Encode codetext only in 128B codeset. |
| CODE_C | `4` | Encode codetext only in 128C codeset. |
| CODE_AB | `3` | Encode codetext only in 128A and 128B codesets. |
| CODE_AC | `5` | Encode codetext only in 128A and 128C codesets. |
| CODE_BC | `6` | Encode codetext only in 128B and 128C codesets. |
