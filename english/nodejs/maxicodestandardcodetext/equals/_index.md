## equals(object) {#equals}

Returns a value indicating whether this instance is equal to a specified [`MaxiCodeStandardCodetext`](../..//maxicodestandardcodetext/) value.

```javascript
equals(obj)
```

| Parameter | Type | Description |
| --- | --- | --- |
| obj | object | An <see cref="MaxiCodeStandardCodetext"/> value to compare to this instance. |

### Return Value

if

obj has the same value as this instance; otherwise, false.
