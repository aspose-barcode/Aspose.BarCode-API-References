## setStructuredAppendModeBarcodesCount(object) {#setstructuredappendmodebarcodescount}

Sets a MaxiCode barcodes count in structured append mode. Count number must be a value between 2 and 8 (maximum barcodes count). Default value: -1

```javascript
setStructuredAppendModeBarcodesCount(value)
```

| Parameter | Type | Description |
| --- | --- | --- |
| value | object | a MaxiCode barcodes count in structured append mode. |
