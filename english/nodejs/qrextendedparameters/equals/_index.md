## equals(object) {#equals}

Returns a value indicating whether this instance is equal to a specified QRExtendedParameters value.

```javascript
equals(obj)
```

| Parameter | Type | Description |
| --- | --- | --- |
| obj | object | An object value to compare to this instance. |

### Return Value

true

if obj has the same value as this instance; otherwise, false.
