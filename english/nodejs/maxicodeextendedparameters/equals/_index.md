## equals(object) {#equals}

Returns a value indicating whether this instance is equal to a specified [`MaxiCodeExtendedParameters`](../..//maxicodeextendedparameters/) value.

```javascript
equals(obj)
```

| Parameter | Type | Description |
| --- | --- | --- |
| obj | object | An Object value to compare to this instance. |

### Return Value

<b>true</b>

if obj has the same value as this instance; otherwise, false.
