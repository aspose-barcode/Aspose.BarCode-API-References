## equals(object) {#equals}

Returns a value indicating whether this instance is equal to a specified Quadrangle value.

```javascript
equals(other)
```

| Parameter | Type | Description |
| --- | --- | --- |
| other | object | An Quadrangle value to compare to this instance. |

### Return Value

true

if obj has the same value as this instance; otherwise, false.
