## AztecSymbolMode enumeration

アステカ シンボル モードを指定します。

```csharp
public enum AztecSymbolMode
```

### 値

| 名前 | 価値 | 説明 |
| --- | --- | --- |
| Auto | `0` | アステカに最適なシンボル (コンパクトまたはフルレンジ) を自動的に選択するように指定します. これはデフォルト値です. |
| Compact | `1` | Aztec のコンパクト シンボルを指定します。 Aztec コンパクト シンボルは、1、2、3、または 4 つのレイヤーのみを許可します。 |
| FullRange | `2` | Aztec のフルレンジ シンボルを指定します。 Aztec フルレンジ シンボルは、1 ～ 32 のレイヤーを許可します。 |
| Rune | `3` | アステカのルーン記号を指定します。 アステカのルーン文字は、一連の小さいながらも明確な機械可読マークです。 0 から 255 までの数値のみを許可します。 |

### 例

このサンプルでは、Aztec シンボル モードを変更し、バーコード イメージを保存する方法を示します。

```csharp
[C#]
using (Aspose.BarCode.Generation.BarcodeGenerator generator = new Aspose.BarCode.Generation.BarcodeGenerator(EncodeTypes.Aztec))
{
    generator.CodeText = "125";
    generator.Parameters.Barcode.Aztec.AztecSymbolMode = AztecSymbolMode.Rune;
    generator.Save("test.png");
}
[VB.NET]
Using generator As New Aspose.BarCode.Generation.BarcodeGenerator(EncodeTypes.Aztec)
    generator.CodeText = "125"
    generator.Parameters.Barcode.Aztec.AztecSymbolMode = AztecSymbolMode.Rune
    generator.Save("test.png")
End Using
```
