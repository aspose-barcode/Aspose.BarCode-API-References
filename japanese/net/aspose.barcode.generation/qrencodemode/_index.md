## QREncodeMode enumeration

QR バーコードのエンコード モード。ラテン記号または数字の場合は CodeTextEncoding = Encoding.UTF8、Unicode 記号の場合は Utf8BOM で Auto を使用することをお勧めします。

```csharp
public enum QREncodeMode
```

### 値

| 名前 | 価値 | 説明 |
| --- | --- | --- |
| Auto | `0` | codetext を非 Unicode 文字セットとしてエンコードします。 Unicode 文字がある場合、コードテキストは CodeTextEncoding. で設定された値でエンコードされます。 |
| Bytes | `1` | codetext をプレーンなバイトとしてエンコードします。 Unicode 文字が検出された場合、その文字は下位バイトから順に 2 バイトとしてエンコードされます。 |
| Utf8BOM | `2` | コードテキストを最初の ByteOfMark 文字で UTF8 エンコーディングでエンコードします。 |
| Utf16BEBOM | `3` | コードテキストを最初の ByteOfMark 文字で UTF8 エンコードでエンコードします。一部のバーコード スキャナーで問題が発生する可能性があります。 |
| ECIEncoding | `4` | ECIEncoding プロパティに設定された値で codetext をエンコードします。一部の古い (2006 年より前の) バーコード スキャナーで問題が発生する可能性があります。 |
| ExtendedCodetext | `5` | FNC1 の最初の位置、FNC1 の 2 番目の位置、およびマルチ ECI モードをサポートする拡張チャネル モード。 |
