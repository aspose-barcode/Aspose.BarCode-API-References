## BaseGenerationParameters.RotationAngle property

バーコード画像の回転角度。度単位で測定されます。たとえば、RotationAngle = 0 または RotationAngle = 360 は回転しないことを意味します。 RotationAngle が 90、180、270、または 0 に等しくない場合、スキャナーが画像を読み取るのが難しくなる可能性があります。 デフォルト値: 0.

```csharp
public float RotationAngle { get; set; }
```

### 例

このサンプルでは、バーコード イメージを作成して保存する方法を示します。

```csharp
[C#]
  using (var generator = new BarcodeGenerator(EncodeTypes.DataMatrix))
  {
      generator.Parameters.RotationAngle = 7f;
      generator.Save("test.png");
  }
```
