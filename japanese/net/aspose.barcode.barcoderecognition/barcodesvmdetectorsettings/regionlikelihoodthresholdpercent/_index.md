## BarcodeSvmDetectorSettings.RegionLikelihoodThresholdPercent property

バーコードを含む可能性がある検出領域のしきい値を設定します。

値 0.7 は、可能性のある領域の下位 70% が除外され、それ以上処理されないことを意味します. 領域の可能性のしきい値は [0.05, 0.9] の間でなければなりません. バーコードの少ない鮮明な画像には高い値を使用してください. 値が小さいと、認識時間が長くなる可能性があります.

public float RegionLikelihoodThresholdPercent { get; set; }
