## CodetextParametersUI.FontMode property

FontMode を指定します。 FontMode が Auto に設定されている場合、フォント サイズは xDimension 値に基づいて自動的に計算されます。 特に AutoSizeMode.Nearest または AutoSizeMode.Interpolation. で FontMode.Auto を使用することをお勧めします。 デフォルト値: FontMode.Auto.

public FontMode FontMode { get; set; }
