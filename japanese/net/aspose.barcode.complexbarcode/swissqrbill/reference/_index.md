## SwissQRBill.Reference property

債権者支払参照を取得または設定します。

SwissQR IBAN、つまり CHxx30000xxxxxx から CHxx31999xxxxx. の範囲の IBAN の参照は必須です。

指定する場合、参照は、有効な SwissQR 参照 (ISR 参照フォームに対応) または ISO 11649 ("RFxxxx") に従った有効な債権者参照 のいずれかでなければなりません。どちらにも書式設定用のスペースが含まれる場合があります.

```csharp
public string Reference { get; set; }
```

### プロパティ値

債権者支払参照。
