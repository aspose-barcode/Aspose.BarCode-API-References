1 次元バーコードの生成時にチェックサムを有効にします。

デフォルトは、チェックサムが必須のシンボルでは Yes とみなされ、チェックサムが可能なだけの場合は No とみなされます。

チェックサムは使用されません: Codabar

チェックサムは可能です: Code39 Standard/Extended, Standard2of5, Interleaved2of5, Matrix2of5, ItalianPost25, DeutschePostIdentcode, DeutschePostLeitcode, VIN

チェックサムは常に使用されます: その他のシンボル
## フィールド

| フィールド | Description |
| --- | --- |
| [DEFAULT](#DEFAULT) | 仕様でチェックサムが必要な場合 - 付加されます。 |
| [NO](#NO) | チェックサムを使用しないでください。 |
| [YES](#YES) | 可能であれば常にチェックサムを使用してください。 |
