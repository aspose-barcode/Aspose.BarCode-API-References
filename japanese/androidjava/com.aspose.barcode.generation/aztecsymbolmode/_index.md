### AUTO {#AUTO}
```
public static final AztecSymbolMode AUTO
```


Aztec の最適なシンボル (COMPACT または FULL-range) を自動的に選択するように指定します。これはデフォルト値です。

### COMPACT {#COMPACT}
```
public static final AztecSymbolMode COMPACT
```


Aztec の COMPACT シンボルを指定します。Aztec COMPACT シンボルは 1、2、3、または 4 層のみ許可します。

### FULL_RANGE {#FULL-RANGE}
```
public static final AztecSymbolMode FULL_RANGE
```


Aztec の FULL-range シンボルを指定します。Aztec FULL-range シンボルは 1 から 32 層まで許可します。

### RUNE {#RUNE}
```
public static final AztecSymbolMode RUNE
```


Aztec の RUNE シンボルを指定します。Aztec Runes は小さくても明確な機械可読マークのシリーズです。0 から 255 の数値のみ許可されます。
