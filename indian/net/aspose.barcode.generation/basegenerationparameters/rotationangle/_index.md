## BaseGenerationParameters.RotationAngle property

बारकोड छवि रोटेशन कोण, डिग्री में मापा जाता है, उदाहरण के लिए RotationAngle = 0 या RotationAngle = 360 का अर्थ है कोई रोटेशन नहीं। यदि RotationAngle 90, 180, 270 या 0 के बराबर नहीं है, तो यह स्कैनर के लिए छवि को पढ़ने में कठिनाई बढ़ा सकता है। डिफ़ॉल्ट मान: 0.

```csharp
public float RotationAngle { get; set; }
```
