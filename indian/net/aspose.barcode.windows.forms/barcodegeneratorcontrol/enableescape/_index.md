## BarCodeGeneratorControl.EnableEscape property

इंगित करता है कि क्या वर्ण "\" को CodeText गुण में एस्केप वर्ण के रूप में समझाता है। Pdf417, DataMatrix, Code128 only के लिए उपयोग किया जाता है यदि EnableEscape सही है, तो "\" को एक विशेष एस्केप कैरेक्टर के रूप में समझाया जाएगा। अन्यथा, "\" सामान्य वर्णों के रूप में कार्य करता है। Aspose.BarCode ASCII नियंत्रण-कोड वर्णों के लिए दशमलव ascii कोड और स्मरक इनपुट करने का समर्थन करता है। उदाहरण के लिए, \013 और \\CR का मतलब CR है।

```csharp
public bool EnableEscape { get; set; }
```
