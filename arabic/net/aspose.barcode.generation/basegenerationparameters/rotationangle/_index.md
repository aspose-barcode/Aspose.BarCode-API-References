## BaseGenerationParameters.RotationAngle property

زاوية دوران صورة الرمز الشريطي ، مقاسة بالدرجة ، على سبيل المثال ، RotationAngle = 0 أو RotationAngle = 360 تعني عدم وجود دوران. إذا لم يكن RotationAngle يساوي 90 أو 180 أو 270 أو 0 ، فقد يؤدي ذلك إلى زيادة صعوبة قراءة الماسح الضوئي للصورة . القيمة الافتراضية: 0.

```csharp
public float RotationAngle { get; set; }
```
