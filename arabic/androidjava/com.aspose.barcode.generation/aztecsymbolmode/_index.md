### AUTO {#AUTO}
```
public static final AztecSymbolMode AUTO
```


يحدد اختيار الرمز الأفضل تلقائيًا (COMPACT أو FULL-range) لـ Aztec. هذه هي القيمة الافتراضية.

### COMPACT {#COMPACT}
```
public static final AztecSymbolMode COMPACT
```


يحدد الرمز COMPACT لـ Aztec. يسمح رمز Aztec COMPACT فقط بـ 1 أو 2 أو 3 أو 4 طبقات.

### FULL_RANGE {#FULL-RANGE}
```
public static final AztecSymbolMode FULL_RANGE
```


يحدد الرمز FULL-range لـ Aztec. يسمح رمز Aztec FULL-range من 1 إلى 32 طبقة.

### RUNE {#RUNE}
```
public static final AztecSymbolMode RUNE
```


يحدد الرمز RUNE لـ Aztec. رموز Aztec Runes هي سلسلة من العلامات الصغيرة ولكن المميزة القابلة للقراءة آليًا. تسمح فقط بقيمة رقمية من 0 إلى 255.
