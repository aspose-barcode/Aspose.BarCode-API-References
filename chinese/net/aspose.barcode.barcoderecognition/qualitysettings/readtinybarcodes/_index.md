## QualitySettings.ReadTinyBarcodes property

允许引擎识别大图像上的小条形码。如果[`AllowIncorrectBarcodes`](../allowincorrectbarcodes)设置为 True，则忽略。默认值:假。

```csharp
public bool ReadTinyBarcodes { get; set; }
```

### 适当的价值

如果为真，则允许引擎识别大图像上的小条形码。
