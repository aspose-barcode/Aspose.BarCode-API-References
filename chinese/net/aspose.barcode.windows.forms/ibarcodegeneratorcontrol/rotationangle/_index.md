## IBarCodeGeneratorControl.RotationAngle property

条码图像旋转角度，以度为单位，例如 RotationAngle = 0 或 RotationAngle = 360 表示不旋转。 如果 RotationAngle 不等于 90、180、270 或 0，可能会增加扫描仪读取图像的难度。
