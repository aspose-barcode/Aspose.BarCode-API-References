Contiene el nivel de confianza del reconocimiento

--------------------

> ```
> This sample shows how BarCodeConfidence changed, depending on barcode type
>  
>  //Moderate confidence
>  BarcodeGenerator generator = new BarcodeGenerator(EncodeTypes.CODE_128, "12345");
>  generator.save("test.png");
>  BarCodeReader reader = new BarCodeReader("test.png", DecodeType.CODE_39_STANDARD, DecodeType.CODE_128);
>      for(BarCodeResult result : reader.readBarCodes())
>      {
>          System.out.println("BarCode Type: " + result.getCodeTypeName());
>          System.out.println("BarCode CodeText: " + result.getCodeText());
>          System.out.println("BarCode Confidence: " + result.getConfidence());
>          System.out.println("BarCode ReadingQuality: " + result.getReadingQuality());
>      }
>  //Strong confidence
>  BarcodeGenerator generator = new BarcodeGenerator(EncodeTypes.QR, "12345");
>  generator.save("test.png");
>  BarCodeReader reader = new BarCodeReader("test.png", DecodeType.CODE_39_STANDARD, DecodeType.QR);
>      for(BarCodeResult result : reader.readBarCodes())
>      {
>          System.out.println("BarCode Type: " + result.getCodeTypeName());
>          System.out.println("BarCode CodeText: " + result.getCodeText());
>          System.out.println("BarCode Confidence: " + result.getConfidence());
>          System.out.println("BarCode ReadingQuality: " + result.getReadingQuality());
>      }
> ```
## Campos

| Campo | Descripción |
| --- | --- |
| [MODERATE](#MODERATE) | Confianza de reconocimiento del código de barras (principalmente códigos de barras 1D) con suma de verificación débil o incluso sin ella. |
| [NONE](#NONE) | Confianza de reconocimiento del código de barras donde el texto del código no se reconoció correctamente o el código de barras se detectó como posible fakeBarCodeExtendedParameters |
| [STRONG](#STRONG) | Confianza de reconocimiento que fue confirmada con códigos BCH como Reed\\u2013Solomon. |

### MODERATE {#MODERATE}
```
public static final int MODERATE
```


Confianza de reconocimiento del código de barras (principalmente códigos de barras 1D) con suma de verificación débil o incluso sin ella. Puede contener algunos errores de reconocimiento en el texto del código o incluso reconocimientos falsos si es baja.

### NONE {#NONE}
```
public static final int NONE
```


Confianza de reconocimiento del código de barras donde el texto del código no se reconoció correctamente o el código de barras se detectó como posible fakeBarCodeExtendedParameters

### STRONG {#STRONG}
```
public static final int STRONG
```


Confianza de reconocimiento que fue confirmada con códigos BCH como Reed\\u2013Solomon. No debe haber errores en el texto del código leído o reconocimientos falsos.
