Las preferencias de BarcodeScanner. Contiene parámetros que influyen en el proceso de reconocimiento y los datos reconocidos. BarcodeScannerPreferences extiende BarcodeRecognitionSettings, la clase BarcodeScannerPreferences contiene elementos específicos de la API de BarcodeScanner y implementa la interfaz Parcelable para intercambiar preferencias entre la actividad cliente y BarcodeScannerActivity. BarcodeScannerPreferences funciona solo dentro de la aplicación cliente a través de BarcodeScannerActivity.
