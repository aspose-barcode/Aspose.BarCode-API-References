## SwissQRBill.Reference property

Obtiene o establece la referencia de pago del acreedor.

La referencia es obligatoria para los IBAN SwissQR, es decir, IBAN en el rango CHxx30000xxxxxx a CHxx31999xxxxx.

Si se especifica, la referencia debe ser una referencia SwissQR válida (correspondiente al formulario de referencia ISR) o una referencia de acreedor válida según ISO 11649 ("RFxxxx"). Ambos pueden contener espacios para formatear.

```csharp
public string Reference { get; set; }
```

### El valor de la propiedad

La referencia de pago del acreedor.
