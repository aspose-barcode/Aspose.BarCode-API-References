## Address.Street property

Obtiene o establece la calle.

Se debe especificar la calle sin número de casa.

Al establecer este campo, se establece el tipo de dirección enStructured a menos que ya sea CombinedElements , en cuyo caso se convierte enConflicting .

Este campo solo se usa para direcciones estructuradas y es opcional.

```csharp
public string Street { get; set; }
```

### El valor de la propiedad

La calle.
