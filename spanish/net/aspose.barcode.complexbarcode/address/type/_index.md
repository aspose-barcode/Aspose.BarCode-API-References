## Address.Type property

Obtiene el tipo de dirección.

El tipo de dirección se configura automáticamente configurando la calle/número de casa o las líneas de dirección 1 y 2. Antes de configurar los campos, el tipo de dirección esIndeterminado . Si se establecen campos de ambos tipos, el tipo de dirección pasa a serContradictorio .
