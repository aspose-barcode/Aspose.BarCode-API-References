## Address class

Dirección del acreedor o deudor.

Puede configurar la calle, el número de casa, el código postal y la ciudad (escribadirección estructurada ) o línea de dirección 1 y 2 (escribaelementos de dirección combinados ). El tipo se establece automáticamente una vez que se establece cualquiera de estos campos. Antes de configurar los campos, el tipo de dirección esindeterminado . Si se establecen campos de ambos tipos, el tipo de dirección pasa a sercontradictorio . El nombre y el código de país siempre deben establecerse a menos que todos los campos estén vacíos.
