## Address.AddressLine2 property

Obtiene o establece la línea de dirección 2.

La línea de dirección 2 contiene el código postal y la ciudad.

Al establecer este campo, se establece el tipo de dirección enCombinedElements a menos que ya sea Structured , en cuyo caso se convierte enConflicting .

Este campo solo se utiliza para direcciones de elementos combinados. Para este tipo es obligatorio.

```csharp
public string AddressLine2 { get; set; }
```

### El valor de la propiedad

La línea de dirección 2.
