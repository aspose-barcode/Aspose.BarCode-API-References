## Address.AddressLine1 property

Obtiene o establece la línea de dirección 1.

La línea de dirección 1 contiene el nombre de la calle, el número de la casa o el apartado de correos.

Al establecer este campo, se establece el tipo de dirección enCombinedElements a menos que ya sea Structured , en cuyo caso se convierte enConflicting .

Este campo solo se usa para direcciones de elementos combinados y es opcional.

```csharp
public string AddressLine1 { get; set; }
```

### El valor de la propiedad

La línea de dirección 1.
