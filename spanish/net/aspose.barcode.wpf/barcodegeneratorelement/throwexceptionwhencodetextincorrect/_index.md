## BarcodeGeneratorElement.ThrowExceptionWhenCodeTextIncorrect property

Solo para códigos de barras 1D. Si el texto en código es incorrecto y el valor se establece en verdadero, se generará una excepción. De lo contrario, el texto del código se corregirá para que coincida con la especificación del código de barras.
