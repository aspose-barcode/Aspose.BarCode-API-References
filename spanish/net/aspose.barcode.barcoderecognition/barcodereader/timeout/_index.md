## BarCodeReader.Timeout property

Obtiene o establece el tiempo de espera del proceso de reconocimiento en milisegundos.

```csharp
public int Timeout { get; set; }
```

### El valor de la propiedad

El tiempo de espera.

### Ejemplos

Este ejemplo muestra cómo evitar bloqueos de reconocimiento con timeount en imágenes grandes

```csharp
[C#]
using (BarCodeReader reader = new BarCodeReader(@"c:\test.png"))
{
    reader.Timeout = 5000;
    foreach (BarCodeResult result in reader.ReadBarCodes())
        Console.WriteLine("BarCode CodeText: " + result.CodeText);
}
[VB.NET]
Using reader As New BarCodeReader("c:\test.png")
    reader.Timeout = 5000
    For Each result As BarCodeResult In reader.ReadBarCodes()
        Console.WriteLine("BarCode CodeText: " + result.CodeText)
    Next
End Using
```
