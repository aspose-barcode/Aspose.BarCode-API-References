## BarCodeConfidence enumeration

Contiene nivel de confianza de reconocimiento

```csharp
public enum BarCodeConfidence
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| None | `0` | Confianza de reconocimiento del código de barras donde el texto del código no se reconoció correctamente o el código de barras se detectó como posible falsificación |
| Moderate | `80` | Reconocimiento de confianza del código de barras (principalmente códigos de barras 1D) con suma de comprobación débil o incluso sin ella. Podría contener algunos reconocimientos erróneos en codetext o incluso reconocimientos falsos si es bajo |
| Strong | `100` | Confianza de reconocimiento que se confirmó con códigos BCH como Reed–Solomon. No debe haber errores en el texto de código leído o reconocimientos falsos |

### Ejemplos

Este ejemplo muestra cómo cambió BarCodeConfidence, según el tipo de código de barras

```csharp
[C#]
//Confianza moderada
using (BarcodeGenerator generator = new BarcodeGenerator(EncodeTypes.Code128, "12345"))
{
    generator.Save(@"c:\test.png");
}
using (BarCodeReader reader = new BarCodeReader(@"c:\test.png", DecodeType.Code39Standard, DecodeType.Code128))
{
    foreach (BarCodeResult result in reader.ReadBarCodes())
    {
        Console.WriteLine("BarCode Type: " + result.CodeTypeName);
        Console.WriteLine("BarCode CodeText: " + result.CodeText);
        Console.WriteLine("BarCode Confidence: " + result.Confidence);
        Console.WriteLine("BarCode ReadingQuality: " + result.ReadingQuality);
    }
}
```
