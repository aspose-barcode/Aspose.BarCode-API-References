## QualitySettings.AllowDecreasedImage property

Permite que el motor reconozca una imagen disminuida como escaneo adicional. Los algoritmos internos del motor seleccionan el tamaño para disminuir. El modo ayuda a reconocer los códigos de barras que tienen ruido y están borrosos pero capturados con alta resolución.

```csharp
public bool AllowDecreasedImage { get; set; }
```

### El valor de la propiedad

Permite que el motor reconozca la imagen disminuida
