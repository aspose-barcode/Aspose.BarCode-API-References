## BarcodeSettings.DetectEncoding property

El indicador que obliga al motor a detectar la codificación de texto en código para conjuntos de códigos Unicode. El valor predeterminado es verdadero.

```csharp
public bool DetectEncoding { get; set; }
```

### El valor de la propiedad

El indicador que obliga al motor a detectar la codificación de texto en código para conjuntos de códigos Unicode

### Ejemplos

Este ejemplo muestra cómo detectar la codificación de texto sobre la marcha si DetectEncoding está habilitado

```csharp
[C#]
using (MemoryStream ms = new MemoryStream())
{
    using (BarcodeGenerator generator = new BarcodeGenerator(EncodeTypes.QR, "Слово"))
    {
        generator.Parameters.Barcode.QR.CodeTextEncoding = Encoding.UTF8;
        generator.Save(ms, BarCodeImageFormat.Png);
    }
    // detecta que la codificación para conjuntos de códigos Unicode está habilitada
    ms.Position = 0;
    using (BarCodeReader reader = new BarCodeReader(ms, DecodeType.QR))
    {
        reader.BarcodeSettings.DetectEncoding = true;
        foreach (BarCodeResult result in reader.ReadBarCodes())
            Console.WriteLine("BarCode CodeText: " + result.CodeText);
    }
    //detectar la codificación está deshabilitada
    ms.Position = 0;
    using (BarCodeReader reader = new BarCodeReader(ms, DecodeType.QR))
    {
        reader.BarcodeSettings.DetectEncoding = false;
        foreach (BarCodeResult result in reader.ReadBarCodes())
            Console.WriteLine("BarCode CodeText: " + result.CodeText);
    }
}
```
