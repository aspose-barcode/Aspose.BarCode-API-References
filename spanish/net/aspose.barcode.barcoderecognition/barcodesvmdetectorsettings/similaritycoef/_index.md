## BarcodeSvmDetectorSettings.SimilarityCoef property

El coeficiente de similitud depende de cuán homogéneos sean los códigos de barras.

Use un valor alto para códigos de barras claros. Use valores bajos para detectar códigos de barras que estén parcialmente dañados o que no se aclaren uniformemente. El coeficiente de similitud debe estar entre [0.5, 0.9]
