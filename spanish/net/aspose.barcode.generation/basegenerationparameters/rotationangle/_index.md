## BaseGenerationParameters.RotationAngle property

Ángulo de rotación de la imagen del código de barras, medido en grados, por ejemplo, RotationAngle = 0 o RotationAngle = 360 significa que no hay rotación. Si RotationAngle NO es igual a 90, 180, 270 o 0, puede aumentar la dificultad para que el escáner lea la imagen. Valor predeterminado: 0.

```csharp
public float RotationAngle { get; set; }
```

### Ejemplos

Este ejemplo muestra cómo crear y guardar una imagen de código de barras.

```csharp
[C#]
  using (var generator = new BarcodeGenerator(EncodeTypes.DataMatrix))
  {
      generator.Parameters.RotationAngle = 7f;
      generator.Save("test.png");
  }
```
