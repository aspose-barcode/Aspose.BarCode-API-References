## DataMatrixParameters.MacroCharacters property

Los valores de los caracteres de macro 05 y 06 se usan para obtener una codificación más compacta en modos especiales. Solo se puede usar con DataMatrixEccType.Ecc200 o DataMatrixEccType.EccAuto. No se puede usar con EncodeTypes.GS1DataMatrix Valor predeterminado: MacroCharacters.None.
