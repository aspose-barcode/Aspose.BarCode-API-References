## QREncodeMode enumeration

Modo de codificación para códigos de barras QR. Se recomienda usar Auto con CodeTextEncoding = Encoding.UTF8 para símbolos o dígitos latinos y Utf8BOM para símbolos Unicode.

```csharp
public enum QREncodeMode
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Auto | `0` | Codifique el texto en código como si fuera un conjunto de caracteres no Unicode. Si hay algún carácter Unicode, el texto del código se codificará con el valor establecido en CodeTextEncoding. |
| Bytes | `1` | Codifica el código de texto como bytes sin formato. Si detecta algún carácter Unicode, el carácter se codificará como dos bytes, el byte inferior primero. |
| Utf8BOM | `2` | Codificar texto en código con codificación UTF8 con el primer carácter ByteOfMark. |
| Utf16BEBOM | `3` | Codificar texto en código con codificación UTF8 con el primer carácter ByteOfMark. Puede haber problemas con algunos lectores de códigos de barras. |
| ECIEncoding | `4` | Codificar texto en código con el valor establecido en la propiedad ECIEncoding. Puede haber problemas con algunos lectores de códigos de barras antiguos (anteriores a 2006). |
| ExtendedCodetext | `5` | Modo de canal extendido que admite la primera posición FNC1, la segunda posición FNC1 y modos ECI múltiples. |
