## AztecSymbolMode enumeration

Especifica el modo de símbolo azteca.

```csharp
public enum AztecSymbolMode
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Auto | `0` | Especifica seleccionar automáticamente el mejor símbolo (Compacto o Rango completo) para Aztec. Este es el valor predeterminado. |
| Compact | `1` | Especifica el símbolo Compacto para Aztec. El símbolo Aztec Compact permite solo 1, 2, 3 o 4 capas. |
| FullRange | `2` | Especifica el símbolo de rango completo para Aztec. El símbolo de rango completo de Aztec permite de 1 a 32 capas. |
| Rune | `3` | Especifica el símbolo de Runa para azteca. Las runas aztecas son una serie de marcas pequeñas pero distintivas legibles por máquina. Solo permite valor numérico de 0 a 255. |

### Ejemplos

Este ejemplo muestra cómo cambiar el modo Símbolo azteca y guardar una imagen de código de barras.

```csharp
[C#]
using (Aspose.BarCode.Generation.BarcodeGenerator generator = new Aspose.BarCode.Generation.BarcodeGenerator(EncodeTypes.Aztec))
{
    generator.CodeText = "125";
    generator.Parameters.Barcode.Aztec.AztecSymbolMode = AztecSymbolMode.Rune;
    generator.Save("test.png");
}
[VB.NET]
Using generator As New Aspose.BarCode.Generation.BarcodeGenerator(EncodeTypes.Aztec)
    generator.CodeText = "125"
    generator.Parameters.Barcode.Aztec.AztecSymbolMode = AztecSymbolMode.Rune
    generator.Save("test.png")
End Using
```
