## BarcodeParameters.EnableEscape property

Indica si se explica el carácter "\" como carácter de escape en la propiedad CodeText. Usado para Pdf417, DataMatrix, Code128 only Si EnableEscape es verdadero, "\" se explicará como un carácter de escape especial. De lo contrario, "\" actúa como caracteres normales. Aspose.BarCode admite la entrada de código ascii decimal y mnemónicos para caracteres de código de control ASCII. Por ejemplo, \013 y \\CR significan CR.
