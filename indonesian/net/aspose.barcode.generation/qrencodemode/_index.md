## QREncodeMode enumeration

Mode penyandian untuk kode batang QR. Disarankan untuk Menggunakan Otomatis dengan CodeTextEncoding = Encoding.UTF8 untuk simbol atau angka Latin dan Utf8BOM untuk simbol Unicode.

```csharp
public enum QREncodeMode
```

### Nilai

| Nama | Nilai | Keterangan |
| --- | --- | --- |
| Auto | `0` | Enkodekan teks kode sebagaimana rangkaian karakter non-Unicode. Jika ada karakter Unicode, teks kode akan dikodekan dengan nilai yang diatur dalam CodeTextEncoding. |
| Bytes | `1` | Mengkodekan teks kode sebagai byte biasa. Jika mendeteksi karakter Unicode apa pun, karakter tersebut akan dikodekan sebagai dua byte, byte lebih rendah terlebih dahulu. |
| Utf8BOM | `2` | Enkode teks kode dengan pengkodean UTF8 dengan karakter ByteOfMark pertama. |
| Utf16BEBOM | `3` | Enkode teks kode dengan pengkodean UTF8 dengan karakter ByteOfMark pertama. Mungkin ada masalah dengan beberapa pemindai kode batang. |
| ECIEncoding | `4` | Enkode teks kode dengan nilai yang ditetapkan di properti ECIEncoding. Mungkin ada masalah dengan beberapa pemindai kode batang lama (pra 2006). |
| ExtendedCodetext | `5` | Mode Extended Channel yang mendukung posisi pertama FNC1, posisi kedua FNC1 dan mode multi ECI. |
