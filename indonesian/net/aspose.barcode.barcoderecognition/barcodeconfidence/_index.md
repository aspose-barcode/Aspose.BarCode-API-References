## BarCodeConfidence enumeration

Berisi tingkat kepercayaan pengenalan

```csharp
public enum BarCodeConfidence
```

### Nilai

| Nama | Nilai | Keterangan |
| --- | --- | --- |
| None | `0` | Keyakinan pengenalan kode batang di mana teks kode tidak dikenali dengan benar atau kode batang terdeteksi sebagai kemungkinan palsu |
| Moderate | `80` | Pengakuan kepercayaan barcode (kebanyakan barcode 1D) dengan checksumm lemah atau bahkan tanpa itu. Bisa mengandung beberapa kesalahan pengenalan di codetext atau bahkan pengenalan palsu jika rendah |
| Strong | `100` | Keyakinan pengakuan yang dikonfirmasi dengan kode BCH seperti Reed–Solomon. Tidak boleh ada kesalahan dalam membaca teks kode atau pengenalan palsu |

### Contoh

Contoh ini menunjukkan bagaimana BarCodeConfidence berubah, bergantung pada jenis kode batang

```csharp
[C#]
// Keyakinan sedang
using (BarcodeGenerator generator = new BarcodeGenerator(EncodeTypes.Code128, "12345"))
{
    generator.Save(@"c:\test.png");
}
using (BarCodeReader reader = new BarCodeReader(@"c:\test.png", DecodeType.Code39Standard, DecodeType.Code128))
{
    foreach (BarCodeResult result in reader.ReadBarCodes())
    {
        Console.WriteLine("BarCode Type: " + result.CodeTypeName);
        Console.WriteLine("BarCode CodeText: " + result.CodeText);
        Console.WriteLine("BarCode Confidence: " + result.Confidence);
        Console.WriteLine("BarCode ReadingQuality: " + result.ReadingQuality);
    }
}
```
