## AustraliaPostCustomerInformationDecoder.Decode method

Dekode Bidang Informasi Pelanggan dari AustraliaPost simbologi. Dapat digunakan untuk interpretasi data yang berbeda dari pengkodean NTable dan CTable. Data disediakan sebagai deretan nilai batang: 0, 1, 2 atau 3.

```csharp
public string Decode(string customerInformationField)
```

| Parameter | Jenis | Keterangan |
| --- | --- | --- |
| customerInformationField | String | Bidang Informasi Pelanggan dikodekan sebagai deretan nilai bilah mentah: 0, 1, 2, atau 3 |

### Nilai Pengembalian

string Bidang Informasi Pelanggan yang didekodekan

### Contoh

Contoh ini menunjukkan cara mendekode data dengan antarmuka AustraliaPostCustomerInformationDecoder

```csharp
[C#]
string[] N_Table = { "00", "01", "02", "10", "11", "12", "20", "21", "22", "30" };
public string Decode(string customerInformationField)
{
    StringBuilder bd = new StringBuilder();
    for (int i = 0; customerInformationField.Length > i; i += 2)
    {
        if (customerInformationField.Length >= i + 2)
        {
            string tmp = customerInformationField.Substring(i, 2);
            for (int j = 0; N_Table.Length > j; j++)
            {
                if (N_Table[j].Equals(tmp))
                {
                    bd.Append(j);
                    break;
                }
            }
        }
    }
     return bd.ToString();
}
```
