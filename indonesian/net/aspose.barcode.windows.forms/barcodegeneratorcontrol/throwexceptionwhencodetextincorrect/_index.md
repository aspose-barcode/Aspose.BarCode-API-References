## BarCodeGeneratorControl.ThrowExceptionWhenCodeTextIncorrect property

Hanya untuk kode batang 1D. Jika teks kode salah dan nilainya disetel ke benar - pengecualian akan dibuang. Jika tidak, teks kode akan diperbaiki agar sesuai dengan spesifikasi kode batang. Pengecualian selalu akan ditampilkan untuk: Simbologi bilah data jika teks kode salah. Pengecualian selalu tidak akan ditampilkan untuk: Simbologi AustraliaPost, SingapurePost, Code39Extended, Code93Extended, Code16K, Code128 jika teks kode salah .
