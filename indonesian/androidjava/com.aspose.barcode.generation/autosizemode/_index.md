### INTERPOLATION {#INTERPOLATION}
```
public static final AutoSizeMode INTERPOLATION
```


Mengubah ukuran kode batang ke ukuran yang ditentukan. Ukuran dapat ditentukan oleh properti ImageWidth dan ImageHeight. Kode batang yang dihasilkan mungkin tidak valid (tidak dapat dibaca) setelah skala.

### NEAREST {#NEAREST}
```
public static final AutoSizeMode NEAREST
```


Mengubah ukuran barcode ke ukuran terdekat terendah yang memungkinkan yang ditentukan oleh properti ImageWidth dan ImageHeight. Mempertahankan rasio aspek default.

### NONE {#NONE}
```
public static final AutoSizeMode NONE
```


Pengubahan ukuran otomatis dinonaktifkan.
