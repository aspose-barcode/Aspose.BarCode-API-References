## EnableChecksum enumeration

Aktivieren Sie die Prüfsumme während der Generierung für 1D-Barcodes.

Der Standardwert wird für Symbologien, die eine Prüfsumme enthalten müssen, als Ja behandelt, als Nein, wenn nur eine Prüfsumme möglich ist.

Nie verwendete Prüfsumme: Codabar

Prüfsumme möglich: Code39 Standard/Extended, Standard2of5, Interleaved2of5, Matrix2of5, ItalianPost25, DeutschePostIdentcode, DeutschePostLeitcode, VIN

Immer verwendete Prüfsumme: Restsymbologien

```csharp

```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Default | `0` | Wenn die Prüfsumme von der Spezifikation verlangt wird, wird sie angehängt. |
| Yes | `1` | Wenn möglich immer Prüfsumme verwenden. |
| No | `2` | Prüfsumme nicht verwenden. |
