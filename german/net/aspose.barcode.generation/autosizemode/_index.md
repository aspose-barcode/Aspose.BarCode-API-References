
### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| None | `0` | Automatische Größenanpassung ist deaktiviert. |
| Nearest | `1` | Ändert die Größe des Barcodes auf die kleinstmögliche Größe, die durch die Eigenschaften BarCodeWidth und BarCodeHeight angegeben ist. Behält das Standard-Seitenverhältnis bei. |
