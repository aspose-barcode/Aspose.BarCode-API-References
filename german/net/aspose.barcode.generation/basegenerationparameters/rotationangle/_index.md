## BaseGenerationParameters.RotationAngle property

Rotationswinkel des Barcode-Bildes, gemessen in Grad, z. B. RotationAngle = 0 oder RotationAngle = 360 bedeutet keine Rotation. Wenn RotationAngle NICHT gleich 90, 180, 270 oder 0 ist, kann dies die Schwierigkeit für den Scanner erhöhen, das Bild zu lesen. Standardwert: 0.

```csharp
public float RotationAngle { get; set; }
```

### Beispiele

Dieses Beispiel zeigt, wie ein Barcode-Bild erstellt und gespeichert wird.

```csharp
[C#]
  using (var generator = new BarcodeGenerator(EncodeTypes.DataMatrix))
  {
      generator.Parameters.RotationAngle = 7f;
      generator.Save("test.png");
  }
```
