## BarCodeGeneratorControl.IsChecksumEnabled property

Aktivieren Sie die Prüfsumme während der Generierung von 1D-Barcodes.

Der Standardwert wird als Ja für Symbologie behandelt, die eine Prüfsumme enthalten muss, als Nein, wenn nur eine Prüfsumme möglich ist.

Prüfsumme möglich: Code39 Standard/Extended, Standard2of5, Interleaved2of5, Matrix2of5, ItalianPost25, DeutschePostIdentcode, DeutschePostLeitcode, VIN, Codabar

Immer verwendete Prüfsumme: Restsymbologie
