## BarCodeReader.ProcessorSettings property

Ruft Einstellungen zur Verwendung von Prozessorkernen ab.

```csharp
public static ProcessorSettings ProcessorSettings { get; }
```

### Beispiele

Dieses Beispiel zeigt, wie ProcessorSettings verwendet wird, um die maximale Multithread-Leistung hinzuzufügen

```csharp
[C#]
//Dies ermöglicht die Verwendung aller Kerne für einen einzelnen BarCodeReader-Aufruf
BarCodeReader.ProcessorSettings.UseAllCores = true;
// Dies ermöglicht die Verwendung der aktuellen Anzahl von Kernen
BarCodeReader.ProcessorSettings.UseAllCores = false;
BarCodeReader.ProcessorSettings.UseOnlyThisCoresCount = Math.Max(1, Environment.ProcessorCount / 2);
[VB.NET]
'Dadurch können alle Kerne für einen einzelnen BarCodeReader-Aufruf verwendet werden
BarCodeReader.ProcessorSettings.UseAllCores = True
'Dies ermöglicht die Verwendung der aktuellen Anzahl von Kernen
BarCodeReader.ProcessorSettings.UseAllCores = False
BarCodeReader.ProcessorSettings.UseOnlyThisCoresCount = Math.Max(1, Environment.ProcessorCount / 2)
```
