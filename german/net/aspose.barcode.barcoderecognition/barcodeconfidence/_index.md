## BarCodeConfidence enumeration

Enthält Erkennungskonfidenzniveau

```csharp
public enum BarCodeConfidence
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| None | `0` | Erkennungszuverlässigkeit des Barcodes, wenn Codetext nicht korrekt erkannt oder Barcode als mögliche Fälschung erkannt wurde |
| Moderate | `80` | Erkennungssicherheit von Barcodes (meist 1D-Barcodes) mit schwacher Prüfsumme oder sogar ohne. Könnte einige Fehlerkennungen in codetext oder sogar gefälschte Erkennungen enthalten, wenn ist niedrig |
| Strong | `100` | Erkennungsvertrauen, das mit BCH-Codes wie Reed-Solomon bestätigt wurde. Es dürfen keine Fehler im gelesenen Codetext oder gefälschte Erkennungen vorhanden sein |

### Beispiele

Dieses Beispiel zeigt, wie sich BarCodeConfidence je nach Barcode-Typ geändert hat

```csharp
[C#]
//Mäßiges Selbstvertrauen
using (BarcodeGenerator generator = new BarcodeGenerator(EncodeTypes.Code128, "12345"))
{
    generator.Save(@"c:\test.png");
}
using (BarCodeReader reader = new BarCodeReader(@"c:\test.png", DecodeType.Code39Standard, DecodeType.Code128))
{
    foreach (BarCodeResult result in reader.ReadBarCodes())
    {
        Console.WriteLine("BarCode Type: " + result.CodeTypeName);
        Console.WriteLine("BarCode CodeText: " + result.CodeText);
        Console.WriteLine("BarCode Confidence: " + result.Confidence);
        Console.WriteLine("BarCode ReadingQuality: " + result.ReadingQuality);
    }
}
```
