## AustraliaPostCustomerInformationDecoder.Decode method

Decodieren Sie das Kundeninformationsfeld aus der AustraliaPost-Symbologie. Kann für unterschiedliche Dateninterpretationen von NTable- und CTable-Codierung verwendet werden. Daten werden als Reihe von Balkenwerten bereitgestellt: 0, 1, 2 oder 3.

```csharp
public string Decode(string customerInformationField)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| customerInformationField | String | Das Kundeninformationsfeld, codiert als Zeile mit Rohbalkenwerten: 0, 1, 2 oder 3 |

### Rückgabewert

die dekodierte Zeichenfolge des Kundeninformationsfelds

### Beispiele

Dieses Beispiel zeigt, wie Daten mit AustraliaPostCustomerInformationDecoder interface decodiert werden.

```csharp
[C#]
string[] N_Table = { "00", "01", "02", "10", "11", "12", "20", "21", "22", "30" };
public string Decode(string customerInformationField)
{
    StringBuilder bd = new StringBuilder();
    for (int i = 0; customerInformationField.Length > i; i += 2)
    {
        if (customerInformationField.Length >= i + 2)
        {
            string tmp = customerInformationField.Substring(i, 2);
            for (int j = 0; N_Table.Length > j; j++)
            {
                if (N_Table[j].Equals(tmp))
                {
                    bd.Append(j);
                    break;
                }
            }
        }
    }
     return bd.ToString();
}
```
