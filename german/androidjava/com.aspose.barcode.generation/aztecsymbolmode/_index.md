### AUTO {#AUTO}
```
public static final AztecSymbolMode AUTO
```


Legt fest, dass automatisch das beste Symbol (COMPACT oder FULL-range) für Aztec ausgewählt wird. Dies ist der Standardwert.

### COMPACT {#COMPACT}
```
public static final AztecSymbolMode COMPACT
```


Gibt das COMPACT‑Symbol für Aztec an. Das Aztec COMPACT‑Symbol erlaubt nur 1, 2, 3 oder 4 Ebenen.

### FULL_RANGE {#FULL-RANGE}
```
public static final AztecSymbolMode FULL_RANGE
```


Gibt das FULL-range‑Symbol für Aztec an. Das Aztec FULL-range‑Symbol erlaubt von 1 bis 32 Ebenen.

### RUNE {#RUNE}
```
public static final AztecSymbolMode RUNE
```


Gibt das RUNE‑Symbol für Aztec an. Aztec‑Runen sind eine Reihe kleiner, aber eindeutiger maschinenlesbarer Markierungen. Sie erlauben nur Zahlenwerte von 0 bis 255.
