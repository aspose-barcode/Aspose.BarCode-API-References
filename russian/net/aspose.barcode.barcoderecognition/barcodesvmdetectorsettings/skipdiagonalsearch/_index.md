## BarcodeSvmDetectorSettings.SkipDiagonalSearch property

Позволяет детектору пропускать поиск диагональных штрих-кодов.

Установка значения false увеличит время обнаружения, но позволит находить диагональные штрих-коды, которые в противном случае могут быть пропущены. Включение диагонального поиска приводит к увеличению времени обнаружения.

public bool SkipDiagonalSearch { get; set; }
