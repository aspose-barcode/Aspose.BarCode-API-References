## BarCodeConfidence enumeration

Содержит уровень достоверности распознавания

```csharp
public enum BarCodeConfidence
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| None | `0` | Достоверность распознавания штрих-кода, если кодовый текст не был правильно распознан или штрих-код был определен как возможный поддельный |
| Moderate | `80` | Уверенность распознавания штрих-кода (в основном одномерных штрих-кодов) со слабой контрольной суммой или даже без нее. Может содержать некоторые неверные распознавания в codetext или даже поддельные распознавания, если низкий |
| Strong | `100` | Достоверность распознавания, подтвержденная кодами BCH, такими как коды Рида-Соломона. Не должно быть ошибок при чтении кодового текста или поддельных распознаваний |

### Примеры

В этом примере показано, как изменялась BarCodeConfidence в зависимости от типа штрих-кода

```csharp
[C#]
//Умеренная достоверность
using (BarcodeGenerator generator = new BarcodeGenerator(EncodeTypes.Code128, "12345"))
{
    generator.Save(@"c:\test.png");
}
using (BarCodeReader reader = new BarCodeReader(@"c:\test.png", DecodeType.Code39Standard, DecodeType.Code128))
{
    foreach (BarCodeResult result in reader.ReadBarCodes())
    {
        Console.WriteLine("BarCode Type: " + result.CodeTypeName);
        Console.WriteLine("BarCode CodeText: " + result.CodeText);
        Console.WriteLine("BarCode Confidence: " + result.Confidence);
        Console.WriteLine("BarCode ReadingQuality: " + result.ReadingQuality);
    }
}

//Сильная уверенность

    }
}
```
