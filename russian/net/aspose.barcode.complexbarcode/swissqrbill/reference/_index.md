## SwissQRBill.Reference property

Получает или задает ссылку на платеж кредитора.

Ссылка обязательна для номеров IBAN SwissQR, то есть IBAN в диапазоне от CHxx30000xxxxxx до CHxx31999xxxxx.

Если указано, ссылка должна быть либо действительной ссылкой SwissQR ( соответствующий форме ссылки ISR) или действительной ссылке кредитора в соответствии с ISO 11649 ("RFxxxx"). Оба могут содержать пробелы для форматирования.

```csharp
public string Reference { get; set; }
```
