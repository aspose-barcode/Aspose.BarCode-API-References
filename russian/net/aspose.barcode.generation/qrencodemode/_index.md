## QREncodeMode enumeration

Режим кодирования штрих-кодов QR. Рекомендуется использовать Auto с CodeTextEncoding = Encoding.UTF8 для латинских символов или цифр и Utf8BOM для символов Unicode.

```csharp
public enum QREncodeMode
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| Auto | `0` | Кодировать кодовый текст как кодировку, отличную от Unicode. Если есть какой-либо символ Unicode, кодовый текст будет закодирован со значением, установленным в CodeTextEncoding. |
| Bytes | `1` | Кодировать кодовый текст как простые байты. Если он обнаружит какой-либо символ Unicode, этот символ будет закодирован как два байта, младший байт первым. |
| Utf8BOM | `2` | Кодировать кодовый текст в кодировке UTF8 с первым символом ByteOfMark. |
| Utf16BEBOM | `3` | Кодировать кодовый текст в кодировке UTF8 с первым символом ByteOfMark. Это могут быть проблемы с некоторыми сканерами штрих-кода. |
| ECIEncoding | `4` | Кодировать кодовый текст со значением, установленным в свойстве ECIEncoding. Это могут быть проблемы с некоторыми старыми (до 2006 года) сканерами штрих-кода. |
| ExtendedCodetext | `5` | Режим расширенного канала, который поддерживает первую позицию FNC1, вторую позицию FNC1 и несколько режимов ECI. |
