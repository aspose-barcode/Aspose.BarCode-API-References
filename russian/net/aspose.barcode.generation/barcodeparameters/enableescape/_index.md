## BarcodeParameters.EnableEscape property

Указывает, объясняет ли символ "\" как escape-символ в свойстве CodeText. Используется только для Pdf417, DataMatrix, Code128 Если EnableEscape имеет значение true, "\" будет объяснен как специальный escape-символ. В противном случае "\" действует как обычный символ.  Aspose.BarCode поддерживает ввод десятичного кода ASCII и мнемоники для символов управляющего кода ASCII. Например, \013 и \\CR означают CR.

public bool EnableEscape { get; set; }
