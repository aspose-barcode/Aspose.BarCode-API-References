## QREncodeType enumeration

Режим выбора QR/MicroQR. Выберите ForceQR для стандартных символов QR, Auto для MicroQR. ForceMicroQR используется для строгой генерации символов MicroQR, если это возможно.

```csharp
public enum QREncodeType
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| Auto | `0` | Режим запускает согласование версии штрих-кода с MicroQR V1 |
| ForceQR | `1` | Режим запускает согласование версии штрих-кода с QR V1 |
| ForceMicroQR | `2` | Режим запускает согласование версии штрих-кода с MicroQR V1 на V4. Если данные не могут быть закодированы в MicroQR, выдается исключение. |
