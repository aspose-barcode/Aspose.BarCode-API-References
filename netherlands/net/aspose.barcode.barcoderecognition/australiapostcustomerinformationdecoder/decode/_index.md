## AustraliaPostCustomerInformationDecoder.Decode method

Decodeer klantinformatieveld uit AustraliaPost-symboliek. Kan worden gebruikt voor andere gegevensinterpretatie dan NTable- en CTable-codering. Gegevens worden geleverd als een rij met staafwaarden: 0, 1, 2 of 3.

### Voorbeelden

Dit voorbeeld laat zien hoe gegevens moeten worden gedecodeerd met AustraliaPostCustomerInformationDecoder-interface

```csharp
[C#]
string[] N_Table = { "00", "01", "02", "10", "11", "12", "20", "21", "22", "30" };
public string Decode(string customerInformationField)
{
    StringBuilder bd = new StringBuilder();
    for (int i = 0; customerInformationField.Length > i; i += 2)
    {
        if (customerInformationField.Length >= i + 2)
        {
            string tmp = customerInformationField.Substring(i, 2);
            for (int j = 0; N_Table.Length > j; j++)
            {
                if (N_Table[j].Equals(tmp))
                {
                    bd.Append(j);
                    break;
                }
            }
        }
    }
     return bd.ToString();
}
```
