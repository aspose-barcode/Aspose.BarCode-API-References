## BaseGenerationParameters.RotationAngle property

Rotatiehoek streepjescodebeeld, gemeten in graden, bijv. RotationAngle = 0 of RotationAngle = 360 betekent geen rotatie. Als RotationAngle NIET gelijk is aan 90, 180, 270 of 0, kan het voor de scanner moeilijker worden om het beeld te lezen. Standaardwaarde: 0.

public float RotationAngle { get; set; }
